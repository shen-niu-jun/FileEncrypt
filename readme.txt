本人纯新手，这个是我的第一个小脚本

这个脚本用于本地自定义密码的单个文件加密解密

没有做环境打包，也没有做自定义背景图片

加密算法也是很简陋的XOR算法
是用C++编译的动态库，源代码也放进/core文件夹了

（不过我觉得用这种加密脚本的有这个就已经够用了）

很多工作都是在AI帮助下完成的



在windows电脑上有node环境和python环境且拥有flask 和flask_cor两个库的情况下，可以直接复制项目后启动start.bat

后端服务跑在本地5000端口，前端服务在本地3000端口

谁会拒绝背景里面的Aliya呢（）

要是不喜欢Aliya的话也可以用自己的图片代替背景图片，然后在/web/style.css里面第11行自己改一下图片地址就可以啦
